## ●台灣電力收費區分為營業用電以及非營業（家用）
其中家用電費更區分為夏季用電以及非夏季用電，此處以預測家用電費為主（非夏季、夏季、橫跨兩季皆可計算） <br> 
power(startdate,enddate,kWh,public,houses)  <br>
startdate 輸入西元年-月-日 <br>
enddate 輸入西元年-月-日 <br>
kWh 輸入家用電費總度數 <br>
public 輸入公共用電總度數 <br>
houses 輸入總家庭數 <br>
